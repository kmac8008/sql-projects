****Cafe Sales Analysis Project with SQL****

**Project Overview**
This project analyzes restaurant order data using SQL to uncover insights about customer behavior, menu performance, and sales trends. The goal is to demonstrate SQL skills commonly used in data analyst roles, including data cleaning, aggregation, filtering, and analytical queries.

https://github.com/kmac8008/sql-projects/blob/main/cafe-sql-project/sql/02_exploratory_analysis%2Bbusiness_questions.sql

**Dataset**
The dataset contains information about restaurant orders including:
Order ID
Item name
Quantity
Price
Order type (Takeaway or Dine-in)
Order date

**Tools Used**
SQL

**Key Business Questions**
This project answers several business questions including:
What are the most ordered menu items?
Which items generate the most revenue?
Are takeaway orders increasing over time?
Which items are popular for takeaway but not in-store?
What day generates the highest sales?

**What I Learned**
Through this project I practiced:
Writing complex SQL queries
Using aggregation and grouping
Performing business-focused data analysis
Translating raw data into actionable insights
